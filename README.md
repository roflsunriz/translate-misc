## このリポジトリについて
このリポジトリには、翻訳済みの記事・論文・メモを統合し、Markdown 化したうえで `mkdocs` で公開するための構成が含まれています。

## ローカル確認
```powershell
mkdocs serve --livereload --dirty
```

## 公開
`main` ブランチへ push すると `.github/workflows/pages.yml` で GitHub Pages 用のサイトをビルド・公開します。
